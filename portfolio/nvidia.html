NVIDIA Summer Internship

Summer 2015

Last summer, I interned at NVIDIA as a Performance Analyst Intern at the company headquarters in Santa Clara, California. I learned from a mentorship program in the first week, then was doing full-time employee work the rest of the twelve-week internship. I received performance requests from different people across the company, other engineers, marketing people, and even executives. I would then run and debug system baselines across different GPUs and CPUs. I wrote automation code to speed up this process as well as new code to test the performance of the newest released video games. I learned a lot.

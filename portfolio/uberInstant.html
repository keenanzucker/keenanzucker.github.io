Uber Instant

Spring 2015

  • GitHub
  • Challenge Post

UberInstant was our project from HackHolyoke in the Spring of 2015. In a team of four, we developed a physical Uber interface. The user first pressed on one of the four buttons, which corresponded to the different type of Uber rides to call (UberX, UberBlack, etc). Then the user spoke their destination. We used Google Voice API to run voice-to-text recognition software to understand what they were saying and call an Uber to their current location, with the destination what they just spoke. We then displayed on the LCD screen the estimated time until the arrival, as well as price of the ride, type of ride, and destination.

Fortunately, Uber's API had a sandbox mode, so we weren't calling actual Ubers to our location when we were testing! We used an Arduino connected to an Adafruit display screen for our hardware, and Python to run the program. This project landed us 2nd place at the hackathon and we got to take home some pretty cool prizes!
